## Draw Tools

 ![](assets/2026-01-16-11-24-36-image.png) The pencil icon in the top right corner opens the drawing tool options.



![](assets/2026-01-16-10-30-34-image.png) Place a marker of your choice at the coordinates that you enter.



![](assets/2026-01-16-10-35-41-image.png) Create a line of specified bearing and distance originating from chosen coordinates.



![](assets/2026-01-16-10-42-33-image.png) Create rings at specified distances from the point of origin. Useful for evacuation operations, searches, and manhunts.



![](assets/2026-01-16-10-36-46-image.png) Opens a window with 5 different points to choose from. Select the desired icon then click on the map to drop the point. Click on the point to access the radial menu.

![](assets/2026-01-16-11-27-34-image.png)

![](assets/2026-01-16-11-05-50-image.png) Delete point.

![](assets/2026-01-16-11-06-34-image.png) Edit location of point by clicking and dragging.

![](assets/2026-01-16-11-06-55-image.png) Lock on point. Note that this feature is more useful for moving points such as aircrafts.

![](assets/2026-01-16-11-07-22-image.png) Open side menu to edit point. From here you can edit name, coordinates, icon style, add notes or attachments, and share with other users through the share button.

![](assets/2026-01-16-11-29-16-image.png)

CloudTAK supports many iconsets (view them by clicking Style ->Select Icon). Note that specialty icon sets may not be supported by other TAK Clients. If they are unsupported they will often be received as a yellow clover icon.

![](assets/2026-01-16-11-30-14-image.png)



![](assets/2026-01-16-10-37-51-image.png) Place points on the map to create a straight line between them. Double click to finish the line. Clicking on the finished line gives you access to the same radial menu as for markers. The Edit button allows you to add accuracy.



![](assets/2026-01-16-10-38-13-image.png) Create any type of shape. Click to put at least two points on the map then double click at the last point to close the shape. Clicking on any shape will give you access to the same radial menu as for markers and lines. You can edit color, opacity, line style, and center coordinates.



![](assets/2026-01-16-10-38-34-image.png) Draw a rectangle in any orientation of your choosing. Drop your first and second point to draw the height of the rectangle. Then use the third point to create the width of the rectangle.



![](assets/2026-01-16-10-42-00-image.png) Draw a circle of any size.



![](assets/2026-01-16-10-39-32-image.png) Draw a sector of any size. First click is the center point, second click is the perimeter.



![](assets/2026-01-16-10-39-57-image.png) Creates a lasso to select any features on your map. Single click to start, single click to finish. After features have been selected, they can be shared, deleted, added to a Data Package, or added to a Data Sync.



![](assets/2026-01-16-10-40-25-image.png) Allows you to import smaller GeoJSONS. Icons imported this way instead of through the Imports tool show up in "Your Features."Â  Each feature is editable so you can change the icon type or location. However, they are features on your map, not an overlay, so you can't toggle them on or off.
